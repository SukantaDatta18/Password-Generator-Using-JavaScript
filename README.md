## ✨ Features
- 🛠️ **Customizable length** (8-32 characters)
- 🔠 **Multiple character types** (Uppercase, Lowercase, Numbers, Symbols)
- 📋 **One-click copy** functionality
- 💪 **Visual strength indicator**
- 🌈 **Clean, modern UI**
- 📱 **Fully responsive** design
